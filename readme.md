
# Python Practice 

## Virtual environment

1. Create
> python -m venv <project_name>

2. Activate
> <project_name>/Scripts/activate 

3. Run main.py to check if VENV is activated or not.  

4. Install packages
> pip install <package_name>

5. Deactivate
> <project_name>/Scripts/deactivate
